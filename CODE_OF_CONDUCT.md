# Code of Conduct

This project adopts the Contributor Covenant, version 2.1, as its code of
conduct. The full text is published at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/
- Plain text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.txt

## In short

We are committed to a welcoming, harassment-free experience for everyone,
regardless of background or identity. We expect participants to be respectful,
act in good faith, accept constructive feedback gracefully, and prioritize what
is best for the community.

Maintainers are responsible for clarifying standards and will take fair
corrective action in response to behavior they deem inappropriate. They have the
right to remove or reject contributions, comments, and other interactions that
do not align with this Code of Conduct.

## Scope

This Code applies in all Olive Bark Logger spaces, including issues, pull requests,
discussions, and any public space where someone is officially representing the
project.

## Reporting

Report concerns privately to the maintainer at **ckellyreif@gmail.com**. All
reports will be reviewed and investigated promptly and fairly, and the privacy
and security of the reporter will be respected.

For the enforcement guidelines and the complete set of expected and unacceptable
behaviors, please read the full Contributor Covenant linked above.
